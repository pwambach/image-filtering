# Image Filtering

WebGL based image filtering app. See [Demo](https://pwambach.github.io/image-filtering/index.html).

## Local development

```sh
git clone <repo> <folder>
cd <folder>

npm install
npm run dev
```

Open in browser on `http://localhost:8000`

## Build

```sh
npm run build
```

Build command outputs into the `./dist` folder
